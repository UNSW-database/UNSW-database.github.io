Data and Knowledge Research Group

The UNSW Data and Knowledge Research Group is world leading for high dimensional data, graph/network data processing, social network analysis, graph databases and artificial intelligence.

Research Interests

Graph Data Processing

Cohesive subgraph computation, social network analysis, attributed graph, spatial-social network, bipartite graphs, heterogeneous information network

Graph Database

Include data mining, data streams, distributed database systems, spatial database systems, and web databases

AI for DB

Machine learning algorithms for graph structured data and database systems, including developing graph neural network architectures and applying machine learning techniques to improve the algorithms in database domain.

Spatial Data Processing

Spatial keyword search, spatial-social data processing
